# AI 기초 OT — 분야 확장판 (푸른 들판 테마)

Vite + React + Tailwind 기반 슬라이드 SPA입니다.

## 로컬 실행
```bash
npm i
npm run dev
```
브라우저에서 http://localhost:5173 접속

## 프로덕션 빌드
```bash
npm run build
npm run preview
```

## Netlify 배포
- **Build command**: `npm run build`
- **Publish directory**: `dist`
- SPA 라우팅은 `netlify.toml`에 설정되어 있습니다.
